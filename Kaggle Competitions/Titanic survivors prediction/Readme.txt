Dans cette compétition, il nous est proposé de prédire à partir d'un jeu de données les passagers du titanic qui ont survécu. L'exercice consiste à 
déterminer le modèle, les features et les hyperparamètres qui permettront d'obtenir le plus grand score d'accuracy. 
Mon meilleur score obtenu est d'environ 78% (exactement 0.77990) avec le RandomForestClassifier de ScikitLearn de Python.
