Le lotto6/49 est un jeu de lotterie canadien où les utilisateurs choisissent une combinaison de 6 nombres dans un ensemble de 49 nombres allant de 1 à 49.
Dans ce notebook, j'évalue les chances de remporter un lot en utilisant quelques techniques de probabilité/statistiques.
Une application de ce travail est que ces chiffres peuvent servir à prévenir les joueurs addictifs que les chances de gagner sont relativement minces d'un point de vue mathématiques.
